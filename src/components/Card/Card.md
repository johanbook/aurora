### Usage

Cards are used for displaying heterogen information.

```js
import Typography from "../Typography";

<Card>
  <Typography>Lorem ipsum</Typography>
</Card>;
```

#### Lists

Cards can be used for listing information.

```js
import Typography from "../Typography";

<div>
  <Card>
    <Typography>Lorem ipsum</Typography>
  </Card>
  <Card>
    <Typography>Lorem ipsum</Typography>
  </Card>
  <Card>
    <Typography>Lorem ipsum</Typography>
  </Card>
  <Card>
    <Typography>Lorem ipsum</Typography>
  </Card>
</div>;
```
